Burp Suite — Penggunaan Dasar di Lab

Panduan konfigurasi singkat untuk menggunakan Burp sebagai proxy pengujian pada browser yang diarahkan ke aplikasi lab.

Langkah Singkat

  1. Set browser untuk menggunakan proxy lokal (mis. 127.0.0.1:8080) hanya untuk host lab.
  2. Gunakan tab Proxy > HTTP history untuk merekam request/response dan analisis header, cookie, dan payload.
  3. Jangan mengarahkan traffic produksi atau situs pihak ketiga melalui proxy ini.

Burp Suite — Web Security Proxy

Burp Suite digunakan untuk analisis aplikasi web: proxy, scanner (komersial), repeater, intruder. Halaman ini membahas arsitektur dan prinsip penggunaan defensif dan riset.

Poin Teknis

  • Arsitektur proxy dan intercept.
  • Automation scanning (gunakan pada lab berlisensi dan jangan terhadap target tanpa izin).
  • Export artefak scan untuk membuat rule WAF/IDS.
