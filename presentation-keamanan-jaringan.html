Ringkasan Presentasi — Keamanan Jaringan

Ringkasan aman dari presentasi proyek Security Monitoring System (Honeypot + SIEM + Network Analysis). Konten ini merangkum konsep dan arsitektur — tidak berisi instruksi menyerang.

Arsitektur Tingkat Tinggi

  • Honeypot (Cowrie) pada VM terisolasi untuk mencatat aktivitas SSH/Telnet.
  • Wazuh Agent di Honeypot mengirim log ke Wazuh Manager untuk analitik & alert.
  • Elastic Stack (Elasticsearch + Kibana) menyimpan dan memvisualisasikan data.
  • Wireshark/Tshark digunakan sebagai alat analisis trafik untuk investigasi.

Komponen & Tujuan Pembelajaran

  • Cowrie: memahami pola brute force dan command behavior di lingkungan aman.
  • Wazuh: korelasi event, pembuatan aturan, dan praktik deteksi insiden.
  • Wireshark: inspeksi paket dan konversasi jaringan untuk forensik.
  • Dashboard: membuat panel metrik serangan, kredensial populer, dan timeline.

Best Practices (Non-aksi)

  • Isolasi jaringan (host-only/VLAN), snapshot VM sebelum eksperimen.
  • Batasi akses dashboard; aktifkan HTTPS dan gunakan kredensial kuat.
  • Rotasi dan kompres log; gunakan log rotation untuk menjaga kapasitas disk.
  • Backup konfigurasi penting dan catat perubahan (change log).

Bacaan Lanjutan

  • Presentasi Keamanan Jaringan (arsip)
  • Cowrie Honeypot (ringkasan)
  • Wazuh SIEM (ringkasan)
  • Snort IDS (ringkasan)

Ringkasan ini hanya untuk pembelajaran etis di lab terkontrol.
