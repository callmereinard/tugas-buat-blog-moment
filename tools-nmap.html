Nmap — Network Mapper

Panduan lengkap Nmap: tipe scan, port specification, kombinasi command yang berguna, dan contoh penggunaan untuk lab terisolasi.

Tool Information

  • Type: Network Scanner
  • Purpose: Port scanning & service detection
  • Difficulty: ⭐ Beginner

Basic Scans

Perintah dasar untuk memulai scanning:

# Simple scan
$ nmap 192.168.1.1

# Scan range
$ nmap 192.168.1.1-254

# Scan subnet
$ nmap 192.168.1.0/24

# Scan from file
$ nmap -iL targets.txt

Scan Types

Berbagai metode scanning untuk keperluan berbeda:

# TCP SYN scan (default)
$ nmap -sS 192.168.1.1

# TCP connect scan
$ nmap -sT 192.168.1.1

# UDP scan
$ nmap -sU 192.168.1.1

# Version detection
$ nmap -sV 192.168.1.1

# OS detection
$ nmap -O 192.168.1.1

# Aggressive scan
$ nmap -A 192.168.1.1

Port Specification

Kontrol port mana yang akan dipindai:

# Specific ports
$ nmap -p 80,443 192.168.1.1

# Port range
$ nmap -p 1-1000 192.168.1.1

# All ports
$ nmap -p- 192.168.1.1

# Top ports
$ nmap --top-ports 100 192.168.1.1

Useful Combinations

Kombinasi perintah yang praktis untuk berbagai skenario:

# Full scan with version detection
$ nmap -sV -sC -p- -T4 192.168.1.1

# Save output
$ nmap -sV -oA scan_results 192.168.1.1
# Creates: scan_results.nmap, .xml, .gnmap

Prinsip Operasional

  • Gunakan scanning yang bertanggung jawab: batasi kecepatan, scope, dan hanya pada aset yang diizinkan.
  • Fingerprinting versi dapat membantu memprioritaskan patching dan mitigasi.
  • Gabungkan hasil dengan sumber threat intel dan CVE untuk analisis risiko.
  • Interpretasi: lihat versi service dan banner untuk menentukan apakah komponen perlu patching atau hardening.
