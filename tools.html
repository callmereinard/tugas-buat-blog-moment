Security Tools & Frameworks

Panduan arsitektur, use-cases, dan rekomendasi penggunaan aman untuk alat standar industri. Halaman ini bertujuan menunjang riset dan operasi keamanan defensif.

Daftar Alat

  • Metasploit (framework)
  • Burp Suite (web security proxy)
  • Nmap (network discovery & scanning)
  • Snort IDS
  • Cowrie Honeypot
  • Wazuh SIEM

Prinsip Penggunaan Aman

  • Jalankan tools hanya di lingkungan terotorisasi/terisolasi.
  • Kombinasikan observability (pcap, logs) untuk analisis pasca-aktivitas.
  • Dokumentasikan konfigurasi, rule sets, dan perubahan untuk reproducibility dan audit.
